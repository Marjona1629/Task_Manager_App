# Task Manager Backend

## Overview
Task Manager Backend is a Java Spring Boot project designed to manage personal tasks. It demonstrates key backend skills including **REST API design, JWT-based authentication, PostgreSQL integration, and Spring Security**. This project is ideal for showcasing your abilities as a junior backend developer.

## Features
- **User Authentication**: Register and login with JWT tokens.
- **Task Management**: Create, read, update, and delete tasks.
- **Secure Endpoints**: All task endpoints are protected using JWT.
- **DTO Mapping**: Uses Data Transfer Objects to separate API models from database entities.
- **Database Integration**: PostgreSQL used with JPA/Hibernate.

## Tech Stack
- Java 17
- Spring Boot 3.x
- Spring Data JPA
- Spring Security
- PostgreSQL
- JWT (io.jsonwebtoken)
- Lombok
